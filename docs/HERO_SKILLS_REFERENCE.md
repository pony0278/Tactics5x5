## Recommended Skill by Playstyle

### Aggressive (進攻型)
- Warrior: **Heroic Leap** — 快速接近敵人
- Mage: **Elemental Blast** — 遠距離高傷害
- Rogue: **Death Mark** — 集火輸出
- Huntress: **Nature's Power** — 爆發傷害
- Duelist: **Elemental Strike** — 單體爆發
- Cleric: **Power of Many** — 團隊增傷

### Defensive (防守型)
- Warrior: **Endure** — 坦傷
- Mage: **Warp Beacon** — 逃跑路線
- Rogue: **Smoke Bomb** — 脫離戰鬥
- Huntress: **Spirit Hawk** — 安全輸出
- Duelist: **Feint** — 反擊威懾
- Cleric: **Ascended Form** — 保命

### Balanced (平衡型)
- Warrior: **Shockwave** — 攻防兼備
- Mage: **Wild Magic** — 群體壓制
- Rogue: **Shadow Clone** — 額外單位
- Huntress: **Spectral Blades** — 範圍傷害
- Duelist: **Challenge** — 控場+輸出
- Cleric: **Trinity** — 全能治療

## Skill Synergies

### With TANK Minion
- **Endure** + Guardian = 超級坦克
- **Challenge** + Guardian = 雙重保護
- **Feint** = 吸引攻擊後反擊

### With ARCHER Minion
- **Death Mark** + Archer Range = 遠距離集火
- **Spirit Hawk** = 雙遠程壓制
- **Wild Magic** = 全場壓制

### With ASSASSIN Minion
- **Heroic Leap** = 雙衝鋒
- **Smoke Bomb** = 配合刺殺
- **Shadow Clone** = 三單位壓制
